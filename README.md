# 🐾 Match Game

Um jogo desenvolvido em **C#** utilizando **Avalonia UI**, com suporte para **Windows** e **Linux**.

Inspirado no primeiro capítulo do livro **Use a Cabeça! C# – 4ª Edição**, este projeto recria o clássico **Match Game** utilizando tecnologias modernas e multiplataforma.

## 🎮 Como jogar

O objetivo é remover todos os pares de animais do tabuleiro.

1. Todos os animais ficam visíveis desde o início da partida.
2. Selecione dois animais iguais.
3. Se o par estiver correto, os dois animais desaparecem do tabuleiro.
4. Continue encontrando os pares até remover todos os animais.
5. Ao finalizar a partida, o tempo total é exibido e você pode iniciar um novo jogo.

## ✨ Funcionalidades

* 🐶 Jogo de associação de pares de animais.
* 🔀 Embaralhamento aleatório dos animais a cada nova partida.
* ⏱️ Cronômetro para registrar o tempo gasto.
* 🎉 Mensagem de conclusão ao terminar o jogo.
* 🔄 Opção para jogar novamente sem reiniciar a aplicação.
* 💻 Interface gráfica desenvolvida com Avalonia UI.
* 🌍 Compatível com Windows e Linux.

## 🛠️ Tecnologias utilizadas

* **C#**
* **.NET**
* **Avalonia UI**
* **JetBrains Rider**
